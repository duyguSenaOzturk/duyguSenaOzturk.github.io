Introduction

RoboVoyager is an innovative teleoperations-robotics project that allows users to remotely control and view the mobile robot's environment through a user-friendly website interface.This project, developed at Hacettepe University, focuses on providing low-latency, reliable connectivity and integrates advanced AI features for object detection & segmentation. This project combines several state-of-the-art technologies to create a comprehensive solution for remote robotics.

The frontend of RoboVoyager is built using the Sveltekit framework, known for its performance and ease of use. The middleware leverages Rosbridge to facilitate communication between the frontend and the backend, which operates on the Robot Operating System (ROS). This architecture ensures a seamless flow of data and commands between the user and the robot, enabling efficient and responsive teleoperations.
