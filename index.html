Introduction

We present the first method capable of photorealistically reconstructing a non-rigidly deforming scene using photos/videos captured casually from mobile phones.

RoboVoyager represents a significant advancement in the field of teleoperations and robotics, offering a robust platform for remote robot control through a web interface. The primary motivation behind the development of RoboVoyager is to address the challenges associated with remote robot operations, such as ensuring reliable connectivity, providing a user-friendly interface, and integrating advanced functionalities like real-time video streaming and AI-driven object detection. This project combines several state-of-the-art technologies to create a comprehensive solution for remote robotics. The frontend of RoboVoyager is built using the Sveltekit framework, known for its performance and ease of use. The middleware leverages Rosbridge to facilitate communication between the frontend and the backend, which operates on the Robot Operating System (ROS). This architecture ensures a seamless flow of data and commands between the user and the robot, enabling efficient and responsive teleoperations.

We show that Nerfies can turn casually captured selfie photos/videos into deformable NeRF models that allow for photorealistic renderings of the subject from arbitrary viewpoints, which we dub "nerfies". We evaluate our method by collecting data using a rig with two mobile phones that take time-synchronized photos, yielding train/validation images of the same pose at different viewpoints. We show that our method faithfully reconstructs non-rigidly deforming scenes and reproduces unseen views with high fidelity.
